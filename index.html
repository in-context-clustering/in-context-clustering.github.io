In-Context Clustering with LLMs


S1. Visualization of Attention

Heatmap visualization of average attention scores of each data point to be clustered. Darker shades indicate higher attention scores. The x-axis and y-axis represent ground truth cluster labels. Attention maps in intermediate layers show similar salient patterns of clusters, while the attnetion in the beginning and end layers show vertical-slash patterns.


S2. Multi-Conditioned Image Clustering

LLMs are capable of producing different clustering given different conditions. The color of the frame outside each image represents the predicted cluster label.


S3. Selective Attention Allocation

Visualization of attention allocation when deciding the cluster for the first image. Highlights indicate high attention scores. LLMs allocates attention differently to different patches across different layers.


S4. Additional results of CD-FSL datasets

ICC achieves comparable performance with k-means and significantly outperformns IC|TC (for more details about the data and baselines, please check sec4.2 of our submission).


S5. Sensitivity to Ordering

Average accuracy on t-distribution (c=2, dim=3) with average std (reflecting sensitivity to ordering) in parentheses. finetuned-llama denotes the llama3.1-8b model finetuned on t-distributed clustering data (sec4.1), and finetuned-llama* denotes finetuning on augmented data with permutation.


S6. Clustering on Attentions

After obtaining the attention matrix as visualized in S1, we normalize the attention matrix and use it as a similarity matrix for spectral clustering. The attention in intermediate layers models the similarity among input data effectively even in zero-shot settings.
